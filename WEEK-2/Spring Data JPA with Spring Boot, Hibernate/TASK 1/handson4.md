HANDS ON 4 – DIFFERENCE BETWEEN JPA, HIBERNATE AND SPRING DATA JPA

JPA (Java Persistence API)
- A specification for data persistence in Java.
- Defines interfaces and standards.
- Does not provide implementation.

Hibernate
- An ORM framework.
- Implements JPA.
- Provides database persistence functionality.

Spring Data JPA
- Built on top of JPA.
- Reduces boilerplate code.
- Provides repository interfaces for database operations.
- Uses Hibernate as the default JPA implementation.
