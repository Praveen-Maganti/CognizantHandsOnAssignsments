Time Complexity
Add: O(1) when adding an employee at the end.
Search: O(n) because employees are checked one by one.
Traverse: O(n) because all employees are visited.
Delete: O(n) because elements need to be shifted after deletion.
Limitations of Arrays
Arrays have a fixed size.
Inserting or deleting elements is slower because other elements must be moved.
Suitable Usage

Arrays are useful when the number of employees is fixed or changes very little and when quick access to employee records by index is needed.
